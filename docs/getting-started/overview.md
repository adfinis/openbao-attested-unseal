# Overview

`openbao-attested-unseal` is not ready to run yet. The current repository
contains the project scaffold, command names, quality gates, release skeleton,
and threat model.

Read [Security: Threat Model](/security/threat-model/) before implementation
work. That page defines the core direction: attested Auto Unseal with TPM as one
backend, not a TPM-only plugin.
